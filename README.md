# Front-End Developer Portfolio

This repository contains a modern React + Tailwind portfolio designed for a Front-End Developer with 4.8 years of experience.

## Features

- Clean landing page with a professional summary drawn from a real resume
- Projects section with sample case studies focused on Figma-to-code, performance, and testing
- Skills section highlighting React.js, Redux Toolkit, Tailwind CSS, Material UI, and testing workflows
- Experience section covering Tata Consultancy Services and Tech Mahindra contributions
- Contact section with LinkedIn, GitHub, and email placeholders

## Install

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```

## Design Notes

- Color palette: deep navy, electric blue, soft gray, and accent teal
- Typography: modern sans-serif with strong heading hierarchy
- Layout: responsive one-page sections for fast scanning and clear navigation
- Style: card-based project blocks, crisp spacing, and accessible contrast
